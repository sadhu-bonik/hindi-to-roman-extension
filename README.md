# Hindi to Roman (Hinglish) Converter Extension

A Google Chrome Extension that converts Hindi text (Devanagari script) into Roman script (Hinglish) on web pages.

## Features
- **Two Modes:** 
  1. **Right-Click:** Convert only selected text (highlights in green).
  2. **Popup Button:** Convert the entire webpage.

## Installation
1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Toggle **Developer Mode** (top right).
4. Click **Load Unpacked**.
5. Select this folder.

## Usage
- **Selection:** Highlight Hindi text, Right-Click, and select "Convert to Roman".
- **Full Page:** Click the extension icon in the toolbar and click "Convert Page".
